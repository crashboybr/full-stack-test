# Deskbookers Task

## Installation Steps

1. Run `npm install`
2. Run `node server.js`
3. Visit http://localhost:3000/

## Routes

`/bookers` params: `?limit=2&offset=0&sort=DESC`

`/booker/{id}`

`/bookings` params: `?limit=2&offset=0&sort=DESC`

`/bookings/{id}`

`/venues` params: `?limit=2&offset=0&sort=DESC`

`/venue/{id}`
